# SiteScope

SiteScope is a Website SEO & Performance Analyzer built using Spring Boot and Jsoup.

## Features

- Analyze any website
- HTTP Status
- Response Time
- Page Title
- Meta Description
- H1 Count
- Images Missing Alt Attribute
- Word Count

## Technologies Used

- Java 21
- Spring Boot 3.5.3
- Jsoup
- HTML
- CSS
- JavaScript
- Maven

## Run the Project

```bash
mvn spring-boot:run
```

Open:

http://localhost:8080

## Test

```bash
mvn test
```
